# GULNcoastal

Gulf Coast Network Monitoring Analysis Tools

## Description

This package provides tools and functions for analyzing coastal monitoring data from Gulf Coast Network National Park units. It includes data processing, analysis, and visualization capabilities for coastal ecosystem monitoring.

## Installation

You can install the development version of GULNcoastal from this repository:

```r
# install.packages("devtools")
devtools::install_local("path/to/GULNcoastal")
```

## Dependencies

This package depends on:
- tidyverse
- ggplot2
- readr
- dplyr
- stringr
- here

```r
library(GULNcoastal)
```
